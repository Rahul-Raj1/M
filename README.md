Movie Recommendation System (Storyline-Based)
📌 Project Overview

The Movie Recommendation System is a content-based filtering application that recommends the top 5 movies based on storyline similarity. The system uses Natural Language Processing (NLP) and machine learning techniques to analyze movie descriptions and suggest similar movies to users.

This project demonstrates the practical implementation of text vectorization (TF-IDF), cosine similarity, and web application development using Streamlit.

🚀 Features

🔍 Recommend Top 5 movies based on storyline similarity

📊 Uses TF-IDF vectorization for feature extraction

📐 Uses Cosine Similarity for similarity calculation

🌐 Interactive web interface built with Streamlit

📁 Dataset generated by scraping movie data from IMDb

💾 CSV-based data storage

🧠 How It Works

Data Collection

Movie titles and storylines are scraped from IMDb using Selenium.

Data is stored in CSV format.

Data Preprocessing

Convert text to lowercase

Remove punctuation and special characters

Remove stopwords

Clean and normalize text

Feature Extraction

Apply TF-IDF (Term Frequency – Inverse Document Frequency)

Convert textual storylines into numerical vectors

Similarity Calculation

Use Cosine Similarity to measure similarity between movies

Rank movies based on similarity score

Recommendation Output

Display Top 5 most similar movies

Results shown in Streamlit interface after clicking “Recommend” button

🛠️ Technologies Used

Python

Selenium (Web Scraping)

Pandas (Data Handling)

NLTK (Text Preprocessing)

Scikit-learn (TF-IDF & Cosine Similarity)

Streamlit (Web Application)
